Sensor Ultrasónico con Arduino

Introducción Objetivo Componentes Código Simulación Conclusión

1. Introducción

Este proyecto utiliza un sensor ultrasónico para medir distancias y activar un sistema de alarma compuesto por un LED y un zumbador. Es un ejemplo práctico de control de salidas basadas en la entrada de sensores con Arduino.

2. Objetivo

Medir la distancia de un objeto utilizando el sensor ultrasónico HC-SR04 y activar alertas visuales y sonoras según un umbral definido, desarrollando habilidades en programación y electrónica básica con Arduino.

3. Componentes

  • Sensor ultrasónico HC-SR04 (pines Trig y Echo)
  • LED de alerta
  • Zumbador (buzzer)
  • Resistencias según necesidad
  • Arduino (Uno, Nano u otro compatible)
  • Cables de conexión y protoboard

4. Código de Arduino

// Pines del sensor ultrasónico
const int trigPin = 9;
const int echoPin = 8;

// Pines de salida
const int alarmaPin = 7;  // LED
const int buzzer = 5;     // Zumbador

void setup() {
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  pinMode(alarmaPin, OUTPUT);
  pinMode(buzzer, OUTPUT);
  Serial.begin(9600); 
}

void loop() {
  long duracion;
  float distancia;

  // Enviar pulso ultrasónico
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);

  // Leer tiempo del rebote del pulso
  duracion = pulseIn(echoPin, HIGH);

  // Calcular distancia en cm
  distancia = duracion * 0.034 / 2;

  Serial.print("Distancia: ");
  Serial.print(distancia);
  Serial.println(" cm");

  // Condición de alerta
  if (distancia > 100) {
    digitalWrite(alarmaPin, HIGH);  // LED encendido
    digitalWrite(buzzer, LOW);      // buzzer apagado
  } else {
    digitalWrite(alarmaPin, LOW);   // LED apagado
    digitalWrite(buzzer, HIGH);     // buzzer encendido
  }

  delay(500); // Tiempo de espera entre mediciones
}

5. Simulación en Tinkercad

El iframe permite interactuar con la simulación directamente desde la página, observar el comportamiento del sensor ultrasónico y las alertas en tiempo real.

6. Conclusión

Este proyecto demuestra la integración de sensores y actuadores en Arduino para crear un sistema de alarma funcional. Permite medir distancias y activar salidas según un umbral definido, facilitando la comprensión de la lógica de control, la programación en Arduino y la electrónica digital básica.

La simulación en Tinkercad proporciona una manera segura y visual de experimentar con el circuito antes de implementarlo físicamente, reforzando el aprendizaje práctico y conceptual del proyecto.
