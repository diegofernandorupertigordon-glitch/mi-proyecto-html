Funciones LCD - Actividad 1 y 2

Introducción Código Arduino Simulación Tinkercad Conclusión

1. Introducción

En esta actividad se implementa un sistema de ingreso mediante un teclado matricial conectado a un Arduino, mostrando la interacción en un display LCD. El circuito permite ingresar una contraseña de 4 dígitos; si la contraseña es correcta, se enciende un LED, mientras que si es incorrecta, se activa un buzzer durante 2 segundos. Además, se visualizan los caracteres ingresados mediante asteriscos en el LCD para mayor seguridad.

2. Código Arduino

#include <LiquidCrystal.h>
#include <Keypad.h>

// Configuración LCD
LiquidCrystal lcd(7, 6, 5, 4, 3, 2);

// Configuración teclado
const byte FILAS = 4;
const byte COLUMNAS = 4;
char teclas[FILAS][COLUMNAS] = {
  {'1','2','3','A'},
  {'4','5','6','B'},
  {'7','8','9','C'},
  {'*','0','#','D'}
};
byte pinesFilas[FILAS] = {A1, A0, 13, 12};
byte pinesColumnas[COLUMNAS] = {11, 10, 9, 8};
Keypad teclado = Keypad(makeKeymap(teclas), pinesFilas, pinesColumnas, FILAS, COLUMNAS);

// LED y buzzer
const int ledPin = A2;
const int buzzerPin = A3;

// Contraseña
String password = "1234";
String ingreso = "";

void setup() {
  lcd.begin(16, 2);
  pinMode(ledPin, OUTPUT);
  pinMode(buzzerPin, OUTPUT);
  
  lcd.clear();
  lcd.setCursor(0, 0);
  lcd.print("Ingrese clave:");
}

void loop() {
  char tecla = teclado.getKey();
  
  if (tecla) {
    if (tecla == '#') {
      if (ingreso == password) {
        lcd.clear();
        lcd.setCursor(4, 0);
        lcd.print("Correcto");
        digitalWrite(ledPin, HIGH);
        digitalWrite(buzzerPin, LOW);
      } else {
        lcd.clear();
        lcd.setCursor(3, 0);
        lcd.print("Incorrecto");
        digitalWrite(ledPin, LOW);
        digitalWrite(buzzerPin, HIGH);
        delay(2000);
        digitalWrite(buzzerPin, LOW);
      }
      ingreso = "";
      delay(1500);
      lcd.clear();
      lcd.setCursor(0, 0);
      lcd.print("Ingrese clave:");
    } else if (tecla == '*') {
      ingreso = "";
      lcd.clear();
      lcd.setCursor(0, 0);
      lcd.print("Ingrese clave:");
    } else {
      ingreso += tecla;
      lcd.setCursor(0, 1);
      for (int i = 0; i < ingreso.length(); i++) {
        lcd.print("*");
      }
    }
  }
}

3. Simulación en Tinkercad

4. Conclusión

La actividad demuestra cómo interactuar con un LCD y un teclado matricial en un Arduino, implementando un sistema de seguridad sencillo mediante contraseña. El uso de LEDs y buzzer proporciona retroalimentación inmediata al usuario según la validez de la clave ingresada, mientras que la visualización mediante asteriscos en el LCD permite mayor privacidad.

Este proyecto refuerza la comprensión de la lectura de entradas digitales, la manipulación de salidas y la integración de múltiples periféricos con Arduino. La simulación en Tinkercad facilita la experimentación práctica sin necesidad de componentes físicos, optimizando el aprendizaje y la verificación del código.
