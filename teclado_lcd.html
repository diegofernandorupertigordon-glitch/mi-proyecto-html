Ingreso por Teclado y Visualización en LCD - Arduino

Introducción Objetivo Componentes Código Simulación Conclusión

1. Introducción

Este proyecto permite ingresar datos mediante un teclado matricial y mostrar la entrada en una pantalla LCD. Se incorpora un sistema de validación de contraseña con alertas visuales (LED) y sonoras (buzzer) en caso de error.

2. Objetivo

Desarrollar un sistema de ingreso por teclado que muestre en pantalla LCD lo ingresado y verifique una contraseña, proporcionando retroalimentación al usuario mediante LED y buzzer, aprendiendo conceptos de entrada/salida digital en Arduino.

3. Componentes

  • Pantalla LCD 16x2
  • Teclado matricial 4x4
  • Arduino Uno/Nano o similar
  • LED indicador
  • Buzzer
  • Resistencias según necesidad
  • Cables y protoboard

Keypad teclado = Keypad(makeKeymap(teclas), pinesFilas, pinesColumnas, FILAS, COLUMNAS);

// Configuración LED y buzzer
const int ledPin = 12;
const int buzzerPin = 11;

String password = "1234";

void setup() {
  lcd.begin(16, 2);
  pinMode(ledPin, OUTPUT);
  pinMode(buzzerPin, OUTPUT);
  digitalWrite(ledPin, LOW);
  digitalWrite(buzzerPin, LOW);
  lcd.clear();
  lcd.setCursor(0,0);
  lcd.print("Ingrese clave:");
}

6. Conclusión

Este proyecto facilita la comprensión de la interacción entre entradas digitales (teclado) y salidas visuales y sonoras (LCD, LED, buzzer), reforzando conceptos de programación y electrónica con Arduino.
